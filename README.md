# bolic
絵文字のみで構成された言語です．

## 使用例
- code
```
ⓧ ☚ ❿ ✖ ❿
ⓨ ☚ ⓧ ➕❽
ⓩ ☚ ❿ ✖ ❹

♨ ❼ ✖ ❿➕❷  ♨ ⓧ ➕❶  ♨ ⓨ  ♨ ⓨ  ♨ ⓧ ➕❿➕❶  ♨ ⓩ ➕❹
♨ ❸ ✖ ❿➕❷  ♨ ⓧ ➕❿➕❾  ♨ ⓧ ➕❿➕❶  ♨ ⓧ ➕❿➕❹ 
♨ ⓨ   ♨ ⓧ   ♨ ❸ ✖ ❿➕❸
```

- result
```
Hello World! 
```

## 命令
### 出力
- char_out: ♨
- num_out: ✒
### ループ 
- while: ♻
- do: ☛
- end: ✅
### 条件分岐
- if: 🤔
- then: ⭕
- else: ❌
- end: ✅
### 四則演算
- +: ➕
- -: ➖
- *: ✖
- /: ➗ 
### 変数操作
assign: ☚
var: (指定しない時)
